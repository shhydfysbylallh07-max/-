```markdown
# لوحة تحكم — تعليمات التشغيل

حل 1 (مستحسن) — واجهة + سيرفر Node.js
1. احفظ الملفات في مجلد المشروع:
   - admin.html
   - admin.css
   - admin.js
   - server.js
   - package.json
2. أنشئ مجلّد uploads وملف db.json فارغ أو اترك السيرفر ينشئهما.
3. ثبت الحزم:
   npm install
4. شغّل السيرفر:
   npm start
5. افتح لوحة التحكم في المتصفح:
   http://localhost:3000/admin.html
   (أو ضع admin.html في مجلد public إذا تريد تقديمها من السيرفر)
6. من الهاتف: افتح نفس العنوان (تأكد أن جهازك على نفس الشبكة أو استخدم ngrok).
   عند رفع صورة، سيُعرض خيار الكاميرا أو اختيار صورة من المعرض.

حل 2 — تجربة محلية سريعة بدون سيرفر
- افتح admin.html مباشرة في المتصفح.
- في أعلى admin.js غيّر toLocal = true
- البيانات (بما فيها الصور كـ base64) ستُخزن في localStorage للمستعرض فقط.

ملاحظات أمان وإنتاج:
- هذا مثال تعليمي سريع. لإنتاج حقيقي:
  - استخدم قاعدة بيانات حقيقية (Postgres, MongoDB, MySQL).
  - تحقق من صلاحيات المستخدم (توثيق/تخويل).
  - تحقق من حجم/نوع الصور قبل الحفظ.
  - استخدم تخزين سحابي (S3) للصور إن كان مطلوباً.
```
